# TOIL (Thailand's Ontology of International Labor)

**Repository for the development of the Thailand's Ontology of International Labor (TOIL).**

* **Authors:** Eugenia Arazo Boa and Michael Brueckner
* **Institution:** Naresuan University International College
* **Project Status:** Phase 1 (Data Gathering, Scoping, and Initial Construction)

## Project Components

1.  **/ontology/**: Contains the core TOIL ontology file (`toil.owl`) and related imports.
2.  **/src/**: Contains the source code for the future online platform.
3.  **/docs/**: Contains project documentation, proposals, and diagrams.
4.  **/data/**: Contains raw data sources used for ontology population (e.g., CSV, XML files from T-MOC, ISCO).
